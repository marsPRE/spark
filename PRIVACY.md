# Privacy Policy

**Last updated: March 13, 2026**

## Overview

SPARKS is a browser-based game that runs entirely on your device. We respect your privacy.

## Data Collection

**We do NOT collect any personal data.**

The game only stores the following information **locally in your browser** using LocalStorage:

- Game settings (audio volume, Morse code speed preferences)
- Save game progress (completed messages, score, reputation)

## Third Parties

- No analytics
- No tracking cookies
- No external servers
- No data sharing

## Your Rights

You can delete all stored data at any time:
- **In-game**: Settings → Reset All Data
- **Browser**: Clear site data for this domain

## Contact

For privacy questions, open an issue on GitHub.
